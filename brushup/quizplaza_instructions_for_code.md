# 実装指示: クイズのひろば（カテゴリ選択のマップ化）

作成: 2026-07-07 ／ 設計: Chat（神田さん承認済み）
方式: パズル島マップと同じ「背景＋%座標の拠点配置」方式

## 目的
クイズのカテゴリ選択画面（現在: 横長カードの縦積みリスト）を、
「クイズのひろば」＝お祭り広場のマップに刷新する。
パズル＝島、クイズ＝ひろば、と場所ごとに景色が変わる（メモ02の場所名と一致させる）。
**変えるのは選択画面の見た目だけ。出題・記録・難易度ロジックは一切変えない。**

## 素材（src/assets/ へ）
- backgrounds/quizplaza-day.webp（やさしい・昼・1600×900・約252KB）
- backgrounds/quizplaza-sunset.webp（ふつう・夕方・約249KB）
- backgrounds/quizplaza-night.webp（むずかしい・夜・約230KB）
- icons/icon_quiz_junban.png（じゅんばん・123ブロック・256px透過）
- icons/icon_quiz_kimari.png（きまりみつけ・虫めがね）
- icons/icon_quiz_nakamawake.png（なかまわけ・かご2つ）
- icons/icon_quiz_robot.png（ロボットめいれい・水色×紫ロボ）
- icons/icon_quiz_yomitori.png（ずのよみとり・フローチャート看板）

## 1. 難易度→背景の対応（パズル島と同じ方式）
- やさしい = quizplaza-day.webp
- ふつう = quizplaza-sunset.webp
- むずかしい = quizplaza-night.webp
MAP_BG 相当の定数で切り替え。**拠点座標は3枚共通**（検証済み・作り直し不要）。

## 2. 拠点の%座標（1箇所の定数に集約）
背景に対する%座標で5カテゴリのアイコンを絶対配置。

| カテゴリ | アイコン | left% | top% |
|---|---|---|---|
| じゅんばん クイズ | icon_quiz_junban | 24.5 | 26.5 |
| きまり みつけ | icon_quiz_kimari | 75.5 | 26.5 |
| なかまわけ クイズ | icon_quiz_nakamawake | 16.5 | 57.0 |
| ロボット めいれい | icon_quiz_robot | 83.5 | 57.0 |
| ずの よみとり | icon_quiz_yomitori | 50.0 | 79.0 |

- 中央(50, 41)は電球モニュメント（装飾・タップ無し。座標はコメントで残す）。
- 座標は目安。実装後にブラウザで照らして微調整（ずれは1〜2ポイントで直る想定）。

## 3. 見せ方（ワールドマップ・パズル島と同じ作法）
- 各拠点: アイコン＋下に短い名前チップ（白地・ひらがな）。
- タップ → ポップアップで「フル名前＋説明＋（もんだい ○こ から 5もん）＋はじめる」。
  現在リストに載っている説明文・問題数・「はじめて/ベスト」表示はポップアップ側へ移す。
- 5カテゴリにロックは無い（全部最初から選べる）。進行の道は描かない。
- 難易度タブ（やさしい/ふつう/むずかしい）は現状のまま上部に維持し、
  切り替えで背景が昼→夕→夜に変わる（パズル島で学んだ視覚言語と統一）。

## 4. やらないこと
- 出題・採点・記録・難易度判定のロジック変更
- 5カテゴリへのロック/順序の追加
- パズル島マップ側の変更

## 5. 完了条件
- 3難易度で背景が昼/夕/夜に切り替わり、5拠点が全難易度で空き地に乗る。
- タップ→ポップアップ→出題へ入り、戻るとひろばに戻る（1階層ずつ）。
- 既存機能不変・verify全PASS。バージョンを上げてデプロイ。

## 6. 実装後のタブレット確認（神田さん）
- iPad横・縦で、3難易度とも5アイコンが空き地に乗っているか。
- ズレは「どの難易度の・どのカテゴリが・時計の何時方向に」で報告
  （座標は3枚共通なので、ズレるなら全難易度で同じ方向のはず）。
