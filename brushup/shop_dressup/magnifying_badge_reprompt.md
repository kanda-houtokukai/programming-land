# 再生成プロンプト: むしめがねバッジ（ピン留め風に作り直し）

作成: 2026-07-07 ／ 理由: 前回生成は「虫眼鏡そのもの」（持ち手つきの単体オブジェクト）
だった。胸(chest)に貼るバッジとして使うには、はぐるまバッジと同じ「ピンバッジ」の
形状に作り直す必要がある。

---

## プロンプト

```
A cute 3D-rendered pin badge shaped like a magnifying glass, for a children's
game item icon, Pixar-style, soft lighting, high saturation, glossy enamel-pin
look — matching the style of a gear-shaped pin badge (flat, rounded, chunky,
meant to be pinned onto a vest pocket, NOT a handheld tool with a long handle).

The magnifying glass shape should be simplified into a badge/pin silhouette:
a round lens shape with a short stubby handle (much shorter than a real
magnifying glass, proportioned like a badge, not an object to hold), warm
gold/bronze rim, a light blue glass tint in the lens, small friendly face or
sparkle optional. Flat-ish enamel-pin thickness (not a fully 3D object with a
long handle sticking out).

Plain white background, no shadow, no text, no characters. Square composition,
centered, slight 3/4 angle. Same style/scale as a matching gear-shaped pin badge.
```

## 生成時の補足（日本語）

「このバッジは、探検家のベストのポケットに"ピンバッジ"として留めるアイテムです。
持ち手のついた道具としての虫眼鏡ではなく、丸いバッジ・ワッペンのような
平たい形にしてください。歯車バッジ（丸いギア型のピンバッジ）と同じ大きさ・
同じ厚み感で揃えてください」

3〜4案出してもらい、以下で選ぶ:
1. 持ち手が短く/なく、バッジらしい平たいシルエットになっている
2. 歯車バッジと並べたとき、同じ「ピンバッジのシリーズ」に見える
3. 虫眼鏡と分かるモチーフ（レンズ＋短い柄）が残っている
