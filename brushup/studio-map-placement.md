## 座標の換算規約（既存の実績値で検算済み・今後の調整もこれに従う）

## アセットの加工規約（既存アセットから実測して確定）

- **縦の余白ゼロ**: 絵が 256px canvas の上下端に接する（既存 building_quiz/home/typing すべて 上0/下0）。
  横は中央寄せ（余白は成り行き）。※初回加工で4%余白を付けており、マップ上で約7%小さく
  表示される不具合があったため修正済み。
- **左右反転済み**: マップ右側の配置のため内向きに反転。副次効果として光源が既存規約に整合
  （既存は全て左光源: quiz +5.9 / home +23.8 / typing +40.1。接地影 drop-shadow(1px 2px 2px) =
  右下に落ちる = 左上光源想定）。反転後: studio +2.7 / sign +10.0 ＝ 左光源で一致。
- スタジオ建物はほぼ正方形(237×238)で既存の tall 建物(縦長)と形が異なるが、
  **tall: true（82%）で表示することを神田さんが実機で承認済み**。
